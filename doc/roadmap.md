# 低コストなAI動画生成環境の構築ロードマップ

## 構築の背景

最近、YouTubeやSNSでハイクオリティなAI動画をよく見かける。自分でも作りたいが、商用サービスは表現の規制やコストの面で不自由なのが実情である。

初期投資を抑えつつ「無規制」な環境を作るため、クラウドの余剰リソースを活用する。構築を通じて、最新のクラウド開発と**「バイブコーディング」から「AIエージェント活用」への変遷**を体験する。

---

## システム要件

| 項目 | 内容 |
|------|------|
| **ターゲット動画** | 2K / 30秒 / 生成10分以内（将来的にVR対応） |
| **コスト** | 初期投資ゼロ、月額数千円ベース（Paperspace $8 + 従量課金） |
| **表現の自由度** | オープンソースモデル利用による表現規制の排除 |
| **開発手法** | 前半: 人間主導のバイブコーディング（構造理解）<br>後半: AIエージェント（Cursor Agent等）による自律構築・最適化 |

---

## システム構成（Master-Worker）

| 役割 | サービス | 課金形態 | 用途 |
|------|----------|----------|------|
| **Master（司令塔）** | Paperspace | 固定費（$8/月） | 開発拠点・API管理・AIとの対話 |
| **Worker（実行機）** | Vast.ai / RunPod | 従量課金 | 高負荷レンダリング・高VRAM処理 |

---

## 用語解説

- **Paperspace（by DigitalOcean）**: 開発拠点（Master）に最適。月額固定のProプラン（$8）でGPUを定額利用可能
- **Vast.ai / RunPod**: 実行機（Worker）に最適。必要な時だけ最強GPUを時間貸しで借りる
- **API連携（WebSocket/JSON）**: GUIで作った設計図をJSONとして書き出し、PythonからWorkerへ送り込む手法。遠隔自動化の標準
- **AIエージェント**: 指示に対して自律的にデバッグや実行を行うツール。開発後半の複雑化に対応

---

## 構築フェーズ

### Phase 1: 開発拠点の固定（バイブコーディング期）

- Paperspace上にDocker環境をデプロイ
- **ポータビリティの確保**: すべての設定を `docker-compose.yml` に集約
- **DoD**: ブラウザからクラウド上のComfyUIにアクセス可能

### Phase 2: GUI習得とAPI自動化（バイブコーディング期）

- **GUIトレーニング**: ノードの繋ぎ方、モデルのロード、画像生成の基本フローをGUI上で練習
- **API書き出し手法**: GUIで組んだワークフローを「API Format JSON」として出力する手順の習得
- **Python連携**: 書き出したJSONをPython経由でWorkerに送信し、結果を受け取る基礎コードの実装
- **I2V対応**: 画像入力を受け付けるノード構成とAPI経由での画像転送ロジックの習得
- **学習理由**: 「GUIでデザインし、APIで量産・遠隔実行する」という分業フローがMaster-Worker構成の核となるため

### Phase 3: 本格生成（エージェント移行期）

- A40（48GB）等のWorkerを使い、2K/30s動画を安定出力
- **I2Vワークフローの最適化**: 画像の構図を維持したまま動画化（SVD, CogVideoX-I2V等）
- **エージェント活用**: 複雑なアップスケール・フレーム補完ロジックのデバッグをAIエージェントに任せる
- **DoD**: 1本の高品質動画がAPI経由で完遂される

### Phase 4: 量産・高速化（エージェント駆動期）

- 複数Workerへの並列割り当て、VR動画対応
- **将来構想（VR対応）**: ステレオ映像生成や等距円筒図法変換ノードの組み込み
- **自律化**: エラー時の自動再試行やリソース最適化をエージェントに完全委ねる

---

## コスト削減の戦略

- **思考は固定費、計算は従量課金**: Paperspaceでじっくりコードを書き、生成時のみ高価なGPUを召喚する
- **ストレージ費ゼロ**: モデルは起動時に外部リポジトリから高速DLする
